## 🔧 技術的な変更内容（詳細）

### 1. コメント返信機能
**ファイル**: `FirestoreService.swift`, `EmotionMapView.swift`

**変更内容**:
- `PostComment` 構造体に `replyToCommentID` と `replyToUserName` を追加
- 投稿者がコメントに返信できる機能を実装
- コメント者も投稿者からの返信に返信可能
- ネスト表示でコメントと返信を見やすく
- 返信通知機能を実装

**影響範囲**: コメント機能を使用する全画面

---

### 2. ミニゲーム報酬の動的調整
**ファイル**: `FirestoreService.swift`, `PrefectureGameMapView.swift`, `MiniGameView.swift`, `GaugeCompleteView.swift`

**変更内容**:
- 10回ごとに報酬が増加する計算式を実装
  ```swift
  let tier = completedCount / 10
  let experience = 100 + (tier * 50)  // 100, 150, 200...
  let postBonus = 1 + (tier / 2)      // 1, 1, 2, 2, 3...
  ```
- UI上で動的に次のレベルの報酬を表示
- リザルト画面で獲得報酬を表示
- ランキング画面に報酬情報を追加

**影響範囲**: ミニゲーム関連の全画面

---

### 3. 投稿回数ボーナスの仕様変更
**ファイル**: `UserService.swift`, `FirestoreService.swift`

**変更内容**:
- 以前: 残り回数を減らす方式（10/10 → 9/10）
- 現在: 上限を増やす方式（10/10 → 11/11）
- `bonusPostLimitKey` で追加の投稿枠を管理
- 日次リセット時に `bonusPostLimit` を0にリセット

**影響範囲**: 投稿制限機能

---

### 4. 管理者のアップデート通知機能
**ファイル**: `FirestoreService.swift`, `AdminDashboardView.swift`, `EmotionPost.swift`, `NotificationsView.swift`

**変更内容**:
- `NotificationType` に `systemUpdate` を追加
- 全ユーザーに一斉通知を送信する機能
- 固定メッセージ: "新しいアップデート！" / "アップデートが来ました！アップデートしてください"
- オレンジ色のメガホンアイコンで表示

**影響範囲**: 管理画面、通知機能

---

### 5. チュートリアルの改善
**ファイル**: `OnboardingView.swift`, `SettingsView.swift`

**変更内容**:
- 地図タブに「悲しい感情が集まってしまうと、、」を追加
- ミニゲームタブに報酬の詳細説明を追加:
  ```
  ⚡ 経験値 +100~
  ➕ 投稿回数 +1~回
  レベルが上がるほど報酬も増えます！
  ```
- チュートリアル再表示のバグを修正
  - キャンセル時に `hasCompletedOnboarding` を変更しないように修正
  - OKボタン押下時のみフラグを変更

**影響範囲**: オンボーディング、設定画面

---

### 6. レスポンシブレイアウトの実装
**ファイル**: 
- `GaugeCompleteView.swift`
- `PrefectureGameMapView.swift`
- `MiniGameView.swift`
- `ProfileView.swift`

**変更内容**:
- `GeometryReader` を使用して画面サイズを取得
- `screenWidth <= 375` (iPhone SE以下) で小さいレイアウトを適用
- フォントサイズ、アイコンサイズ、パディング、スペーシングを調整
- アバターサイズやボタンサイズを画面に応じて最適化

**対応デバイス**: iPhone SE (第2世代/第3世代) 以下の小型端末

**調整項目**:
- フォントサイズ（例: 40px → 28px）
- アイコンサイズ（例: 60px → 44px）
- パディング（例: 24px → 16px）
- スペーシング（例: 30px → 20px）
- アバターサイズ（例: 100px → 80px）

**影響範囲**: 主要な4画面のUI
