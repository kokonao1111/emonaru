# エモナル

- このアプリは自分の感情をその場で投稿共有することができるIOSアプリです
- 学内のコンテストのため開発しました
- 見えない感情を可視化することでいつもと違う新しいSNSになっています
- アップルストアに公開中です

# 機能

- 感情を投稿
- 投稿が地図上で表示される
- 知らない人の感情を可視化その場で今どんな感情を持っているのかがわかる
- 感情によってコミニケションがとれつながることができる
- 感情を使ったミニゲーム機能あり



# 使用技術

- 言語
- Swift（メイン実装、hyu/src）

- SwiftUI
- MapKit / CoreLocation（地図・位置情報）
- UserNotifications（通知）
- PhotosUI / Photos（写真選択）

- Cloud Firestore（投稿・ユーザー・通知データ）
- Firebase Storage（画像アップロード系）
