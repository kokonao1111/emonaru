# エモナル

このアプリは自分の感情をその場で投稿共有することができるIOSアプリです
見えない感情を可視化することでいつもと違う新しいSNSになっています
アップルストアに公開中です

Users share emotions without words.
There are no profiles, no comments, no likes, and no followers.

Each post is only an emotion level (-5 to +5) with abstract visuals.
Posts fade and disappear.

Design principles:
- No explanation
- No permanence
- No comparison
- Calm, minimal UX
